This is essentially Pagedown with some slight modifications.
http://code.google.com/p/pagedown/wiki/PageDown

Modifications:
- Work with configurable pretty code processor,
- Opened the configuration object so it can pull configs down into processing channels.
- Fixed some character encoding issues.
- Made MarkdownExtra's a little easier to get running.
- Many subtle problem children were placed in straight-suits.

Example
var markdownC = require("./markdown").Converter;
var converter = new markdownC();
var markdownExtra = require("./markdown-extra");
markdownExtra.init(converter);

var myHtml = converter.makeHtml( decodeHtmlEntities(text), opts );
